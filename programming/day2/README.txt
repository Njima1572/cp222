This directory contains the java source files for a data structure representing a 2 dimensional grid with only integer coordinates.

Files:
IIntPoint2D.java - An interface for a 2D point with integer coordinates
IIntGrid2D.java     - An interface for a 2D grid
Test.java        - A program to test a 2D grid implementation
IntPoint2D.java	 - A class for a 2D point with integer coordinates
IntGrid2D.java	 - A class for a 2D grid
